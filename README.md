## 🚀 Instalação e Configuração

### Pré-requisitos

* **Python 3.13+**
* **MetaTrader 5 Terminal** (instalado e com sessão iniciada na corretora).
* **Compilador C++17 / CMake** (apenas se necessitar de recompilar a `asi_core.dll`).
* **Java JDK 11+** (para executar os *Daemons* do subconsciente).

### Passo a Passo

1. **Clonar o Repositório**
```bash
git clone [https://github.com/seu-usuario/DubaiMatrixASI.git](https://github.com/seu-usuario/DubaiMatrixASI.git)
cd DubaiMatrixASI

```


2. **Configurar o Ambiente Virtual Python**
```bash
python -m venv .venv
source .venv/bin/activate  # Em sistemas Linux/macOS
# No Windows utilize: .venv\Scripts\activate

pip install -r requirements.txt

```


3. **Configuração da Bridge MQL5**
* Copie o ficheiro `DubaiMatrixASI_HFT_Bridge.ex5` (ou `.mq5` para compilar) da pasta `mql5/` para o diretório `MQL5/Experts/` do seu terminal MetaTrader 5.
* Arraste o *Expert Advisor* para o gráfico desejado e certifique-se de que a opção "Permitir Algorithmic Trading" está ativa.


4. **Configuração de Variáveis (Configuração Inicial)**
* Ajuste as suas chaves de API, credenciais e limiares de risco nos ficheiros dentro de `config/` (ex: `settings.py`, `omega_params.py`).
